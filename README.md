# Chaithanya K Portfolio Website

## Introduction

Welcome to the portfolio website of Chaithanya K! This website serves as a personal portfolio to showcase projects, education, and provide links to social media profiles. The website is built using HTML, CSS, and JavaScript.

## Project Structure

The project is organized into the following files and directories:

```
ChaithanyaK-Portfolio/
├── index.html
├── project.html
├── edu.html
├── social/
│   ├── email.webp
│   ├── github.webp
│   ├── instagram.webp
│   └── linkedin.webp
├── style.css
└── script.js
```

## Files and Directories

- `index.html`: The main landing page for the portfolio.
- `project.html`: The page dedicated to showcasing projects.
- `edu.html`: The page dedicated to displaying educational background.
- `social/`: Directory containing social media icon images.
- `style.css`: The CSS file for styling the website.
- `script.js`: The JavaScript file for any interactive functionality.

## Instructions

### Prerequisites

To view and interact with the website, you will need a web browser (e.g., Google Chrome, Mozilla Firefox, Safari).

### Running the Website

1. **Clone the Repository:**
   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the Project Directory:**
   ```bash
   cd ChaithanyaK-Portfolio
   ```

3. **Open the `index.html` File in a Browser:**
   - You can open the file directly by double-clicking it.
   - Alternatively, you can open it from the command line:
     ```bash
     open index.html  # macOS
     start index.html  # Windows
     xdg-open index.html  # Linux
     ```

### Directory Structure Details

- **`index.html`:**
  The main homepage of the portfolio. It introduces Chaithanya K, provides a brief description, and links to social media profiles.

- **`project.html`:**
  This page is intended to showcase various projects. You can add descriptions, links, and images of your projects.

- **`edu.html`:**
  This page is intended to display the educational background. You can add details about schools, degrees, and any notable achievements.

- **`social/`:**
  This directory contains icons for social media platforms (Instagram, GitHub, LinkedIn, and Email).

- **`style.css`:**
  The main CSS file that styles the website. It includes styles for layout, fonts, colors, and responsive design.

- **`script.js`:**
  The JavaScript file for adding interactivity. Currently, it's empty but can be used for future enhancements.

## Customization

You can customize the content and styles of the website by editing the HTML, CSS, and JavaScript files. Here are some suggestions for customization:

- **Update Content:** Add your personal information, projects, and educational background to the respective HTML files.
- **Enhance Styles:** Modify the `style.css` file to change the look and feel of the website.
- **Add More Pages:** Create additional pages if you want to include more sections like blogs, achievements, etc.
